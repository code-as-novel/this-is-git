### Configuration

```

git config user.name

git config --global user.email "sang@gmail.com"
git config user.email
```

## command line

- touch : create empty file
```
touch "filename"
```

- start / open : opens window 
```
start .    # windows
open .     # mac
```

- pwd : print working directory
```
pwd
```
